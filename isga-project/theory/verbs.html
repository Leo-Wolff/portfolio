Verbs and tenses

The Dutch language uses 3 main tenses. But before we will learn when to use each one, it is important to know how to find the stem of a verb so you can conjugate it.

Stem

To find the stem of a verb you have to remove the -en after it.

Example:
Werken → werk
To work → work

There are some verbs where the pronunciation changes when you create the stem. This can be fixed by adding things such as an extra vowel.
Lopen → loop
To walk → walk

You also have cases where there is a double consonant, this looks weird and thus it gets dropped:
Hakken → hak
To chop → chop

Pronouns

Before we get into actually conjugate the stems there is one more thing that is important to know: pronouns.

Dutch English
Ik I
Jij You
Hij/zij/u He/she/they
Wij We
Jullie You
Zij They

Incompleted Present Tense

The incompleted present indicates an action that is currently happening and hasn’t completed yet.

Rule Example
Ik [stem] Ik werk
Jij [stem] + t Jij werkt
Hij/zij/u [stem] + t Hij/zij/u werkt
Wij [stem] + en Wij werken
Jullie [stem] + en Jullie werken
Zij [stem] + en Zij werken

Completed Present Tense

The completed present indicates an action that is currently happening and has been completed. This tense only has one conjugation.

The completed present tense has a second verb that gets added (also known as a help-verb).
The help-verb gets conjugated in its place. This help verb can be hebben (have) and worden (become).

As for the completed present verb, the following structure applies:
Ge- [stem] + d

Example:
Wij hebben geleerd.
We have learned.

Note: words that already have a prefix, do not get ge- at the start.
Example:
Het wordt herhaald.
It is being repeated.
You will also find that completed present tenses can end with a -t, this is explained in strong verbs.

Incompleted Past Tense

The incompleted past tense indicates an action that has already happened, but doesn’t mention it being completed.

Rule Example
Ik [stem] +te Ik werkte
Jij [stem] + te Jij werkte
Hij/zij/u [stem] + te Hij/zij/u werkte
Wij [stem] + ten Wij werkten
Jullie [stem] + ten Jullie werkten
Zij [stem] + ten Zij werkten

Note: if you want a completed past tense, you simply conjugate the help-verb to this tense.
Example:
Jij had gespeeld.
You had played.

Irregular verbs

You may have seen it in the previous section, but there are verbs that don’t follow these rules. These verbs are a part of many.

The irregular verbs are all verbs that end with the following letters in their stem:
x, t, k, f, s, ch, or p

The Dutch do use a simple memory aid to help remember all of these:
Het sexy fokschaap
The sexy breeding sheep

Note: there are many different version to this as well, het fokschaap, het fokschip (breeding ship), etc.

If any letters of this sentence are at the end of the stem of a word the following conditions apply:

  • The completed present tense uses a -t in the end instead of -d.
  • The incompleted past tense uses a -d instead of the -t (this also contains for verbs that end with a d in their stem).
